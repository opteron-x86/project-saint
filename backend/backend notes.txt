These are the backend functions for AWS Lambda to perform processing, enrichment, API endpoints, and datamodels. Each uses the datamodel layer and environment variables to connect to the Aurora Serverless v2 PostgreSQL database in RDS. Dependencies are provided by a dependency layer. The API handler function is triggered by API Gateway routes. The Processing functions are triggered by uploads into an S3 bucket for a specific ruleset. The frontend routes calls through the API Gateway to query the API endpoints.
